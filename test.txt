1. Coach Update Password
Endpoint:
PUT /coach/update-password

Example Request (JSON):

{
  "email": "coach1@email.com",
  "currentPassword": "oldcoachpass",
  "newPassword": "newcoachpass"
}

2. Admin Update Password
Endpoint:
PUT /admin/update-password

Example Request (JSON):

{
  "email": "admin@email.com",
  "currentPassword": "oldadminpass",
  "newPassword": "newadminpass"
}

Update password:
PUT /player/update-password

{
  "email": "player@email.com",
  "oldPassword": "oldpass",
  "newPassword": "newpass"
}
Reset password:
PUT /player/reset-password

{
  "email": "player@email.com",
  "mobile": "1234567890",
  "newPassword": "newpass"
}
